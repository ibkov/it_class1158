ХАКАТОН 1-11 МАЯ 2020

ТЕМА ХАКАТОНА: «ДИСТАНЦИОННОЕ ОБУЧЕНИЕ»

Для альтернативной защиты темы по программированию «Функции» в период с 1 по 11 мая вы можете создать любой общедоступный продукт (web-приложение, web-сайт, desktop application(для доступных платформ), игра, web-игра, мобильное приложение и.т.д.) соответствующий тематике хакатона.

КРИТЕРИИ ОЦЕНИВАНИЯ:

  1. Продукт должен быть общедоступным для любого участника образовательного процесса
  2. Продукт должен решать, какую-либо проблему связанную с дистанционной формой образования
  3. Продукт должен быть полезен для целевой аудитории
  4. Продукт должен содержать интуитивно-понятный интерфейс или инструкцию по применению.
  5. Продукт должен быть продуктом Opensource (иметь возможность изменять функционал сторонними разработчиками)
  6. Код продукта должен быть доступен на открытых платформах github или др.

ТЕХНИЧЕСКАЯ СТОРОНА ОЦЕНИВАНИЯ:

  1. В коде продукта должно использоваться не менее 5 функций написанные самостоятельно.
  2. Функции должны иметь возможность принимать любое количество позиционных аргументов
  3. Название функций должны соответствовать функциональности.
  4. Минимум 1 функция должна иметь возможность принимать любое количество собственных именованных аргументов
  5. Минимум в 1 функции должны быть использованы значения по умолчанию
  6. Минимум 1 функция должна использовать значение возвращаемое другой самостоятельной функцией
  7. Не одна функция не должна менять значения глобальных переменных

ПРАВИЛА УЧАСТИЯ:

  1. В срок до 2 мая (включительно) до 23.59 заполнить форму с регистрацией команды (не более 3 человек) и продукта нажав на кнопку "ЗАРЕГИСТРИРОВАТЬ КОМАНДУ"
  2. 4 мая в течении дня, минимум один человек из команды (возможно команда целиком) пройти первый этап защиты идеи продукта.
  3. 10 мая в течении дня пройти второй этап защиты готового продукта.
  4. В период с 11-15 мая представить, в назначенное команде время, продукт перед аудиторией (дистанционно), предоставить возможность использования продукта.
